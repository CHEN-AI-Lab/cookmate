# CookMate 设计文档

## 设计原则

1. **食欲驱动** — 色彩温暖，图片诱人，让用户看了就想做饭
2. **极简操作** — 3 步以内完成核心操作
3. **移动优先** — 手机是主要使用场景
4. **中文优先** — 先做中文用户，再支持英文

## 配色方案

- 主色: #FF6B35 (暖橙 — 食欲色)
- 辅色: #2D9CDB (清爽蓝)
- 背景: #FFF8F0 (米白)
- 文字: #2D3436 (深灰)
- 成功: #27AE60 (翠绿 — 健康)
- 卡片: 白色 + 圆角 16px + 轻阴影

## 字体

- 标题: Noto Sans SC (Google Fonts)
- 正文: Inter
- 强调: 用 emoji 🍳🥗🍝 配合文字

## 页面设计要点

### 首页
- Hero: 大标题 "今天吃什么？CookMate 帮你决定"
- 副标题: "告诉我你有什么食材，3秒生成菜谱"
- Demo 区: 输入"鸡肉、西兰花、米饭" → 实时展示 AI 生成
- 3 步使用流程卡片
- 定价区 → CTA

### Dashboard
- 顶部: 今日推荐菜谱卡片
- 中间: 本周计划概览
- 底部: 快速操作（添加食材 → 生成菜谱）

### AI 食谱页
- 输入区: 文本输入框（食材用 tag 形式展示）
- 筛选: 菜系/时间/难度
- 结果: 卡片列表，每张含图片占位 + 菜名 + 时间 + 卡路里
- 点击展开: 详细步骤

### 周计划
- 7 天横排日历
- 每天显示三餐
- 点击替换 → 弹出 AI 推荐
- 底部: "生成购物清单" 按钮

## 响应式

- Mobile: 单列，底部导航
- Desktop: 侧边栏 + 内容区
- Tablet: 自适应
