# CookMate 产品规格

## 一句话

**CookMate 帮你 3 秒决定 "今天吃什么"。**

## 用户画像

- 25-45 岁，城市上班族
- 每天为 "吃什么" 发愁
- 想吃得健康但没时间规划
- 有饮食习惯但缺乏执行力
- 会做饭但不想花时间想菜单

## 核心流程

```
① 用户打开 CookMate
② 告诉 AI 有什么食材/想吃什么
③ AI 生成菜谱（含步骤、时间、营养）
④ 用户选择 → 加入周计划
⑤ 自动生成购物清单
⑥ 去超市买菜 / 立刻做
```

## MVP 功能清单

### Phase 1 — 核心闭环（本版本）

1. **AI 食材→食谱生成**
   - 输入食材列表 → AI 推荐可做的菜
   - 按菜系/口味/时间筛选
   - 生成结果含：菜名、配料、步骤、用时、卡路里

2. **周计划**
   - 拖拽式 7 天日历
   - AI 一键生成整周计划
   - 手动替换某一天的菜

3. **智能购物清单**
   - 根据周计划自动汇总
   - 按超市分区归类（蔬菜/肉/调料等）
   - 可勾选已买

4. **食材管理（简单版）**
   - 手动添加冰箱里的食材
   - 标记过期/用完

### Phase 2 — 付费功能（部分实现）

## 技术边界

- 支持中文 + 英文菜谱
- AI 生成响应时间 < 5s
- 菜谱数据不存本地，高频请求走缓存
- 支付走 Stripe，支持支付宝（后期）
