# CookMate 食材风控清单

本文件定义了不可用于 AI 菜谱生成的食材/词汇。
用于 API 路由中的输入验证，拦截风险内容。

## 分类说明

### 1. 非食用物品 (NON_FOOD)
非食材类物品，输入这些词说明用户乱填。
- 矿物/金属：石头、沙子、泥土、铁、铜、铝、钢、钉子、螺丝、水泥、玻璃
- 日常用品：塑料、纸、布、橡胶、胶水、电池、绳子、木头
- 建材/工具：油漆、涂料、胶带、钉子、螺丝、铁丝
- 其他杂物：树叶、树皮、草、花瓣（非食用花卉）

### 2. 有毒有害物质 (TOXIC)
化学物质、有毒物、清洁剂等，绝对不能出现在食品中。
- 化工原料：甲醇、乙醇（工业）、工业酒精、甲醛、苯、丙酮
- 清洁剂：洗衣粉、洗洁精、漂白水、洁厕灵、消毒液、84消毒液
- 农药：农药、杀虫剂、除草剂、百草枯、敌敌畏
- 有毒植物：毒蘑菇、毒草、夹竹桃、曼陀罗、断肠草、乌头
- 重金属：汞、水银、铅、镉、砷

### 3. 国家保护动植物 (PROTECTED)
受中国法律保护的野生动植物，严禁食用。
- 一级保护动物：大熊猫、金丝猴、东北虎、雪豹、藏羚羊、扬子鳄、中华鲟
- 二级保护动物：黑熊、穿山甲、天鹅、猫头鹰、海龟
- 其他保护：鲸鱼、鲨鱼（鱼翅）、海马、珊瑚
- 保护植物：红豆杉、银杏、珙桐、野生人参、雪莲（部分受保护）
- 通用词：保护动物、野生动物、国家保护

### 4. 毒品及违禁品 (DRUGS)
中国法律明令禁止的毒品和违禁药物。
- 毒品：海洛因、冰毒、大麻、可卡因、吗啡、鸦片、摇头丸、K粉
- 违禁药物：罂粟、罂粟壳、麻黄草
- 精神药物：LSD、神仙水、开心水

### 5. 非法/不道德食材 (ILLEGAL)
涉及虐待动物、非法猎捕、濒危物种等。
- 猫、狗（伴侣动物，中国多地已禁止食用）
- 蝙蝠、果子狸（野生动物+疫病风险）
- 活体动物（活吃、虐待动物相关）

### 6. 虚构/不存在食材 (FICTIONAL)
现实世界不存在的虚构生物。
- 神话生物：恐龙、龙、凤凰、独角兽、麒麟、美人鱼
- 科幻生物：外星人、异形
- 传说生物：年兽、年怪

### 7. 违禁添加剂 (ADDITIVES)
中国食品安全法禁止使用的食品添加剂。
- 吊白块、苏丹红、三聚氰胺、工业明胶、硼砂
- 甲醛（防腐）、工业盐、罂粟壳
